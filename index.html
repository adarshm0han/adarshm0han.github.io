Selected Projects

Agentic Career Assistant (Knowledge Graph + Multi-Agent)

Built an agentic career assistant that parses resumes and job descriptions into a Neo4j knowledge graph and uses LangGraph-based multi-agent workflows to generate explainable candidate–job matches.

  • Knowledge graph entities: Candidate, Job, Skill, Domain with typed relations
  • Hybrid scoring using Jaccard overlap + embedding-based semantic similarity
  • Transparent outputs with evidence-based match justification

WebSecArena: Benchmarking Web Agents Under Security Threats

Co-developed a benchmark to evaluate LLM-powered web agents under realistic web security threats including prompt injection, phishing, social engineering, malicious redirects, and drive-by downloads.

  • Evaluation pipeline built with BrowserGym and AgentLab
  • Metrics: Attack Success Rate (ASR), Safe Task Success (STS), Disrupted Task Rate (DTR)
  • Findings expose trade-offs between task completion and security robustness

NYC Taxi Graph Analytics Pipeline

Designed and deployed a scalable real-time graph analytics pipeline for NYC Yellow Cab data.

  • Neo4j graph with Location nodes and TRIP relationships
  • PageRank and BFS implemented using Neo4j GDS
  • Streaming ingestion via Kafka + Kubernetes
  • Performance: ~70% CPU and ~65% RAM utilization with scaling headroom
