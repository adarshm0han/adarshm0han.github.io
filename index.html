Selected Projects

News Sentiment · DistilBERT vs VADER

Lightweight, serverless news‑sentiment service that compares a rules‑based lexicon model (VADER) against a fine‑tuned transformer (DistilBERT). Co‑built the app while leading the ML side: training, fine‑tuned DistilBERT, served predictions via AWS Lambda calling the Hugging Face Inference API, and collaborated on VADER/AWS integration. Achieved 0.81 macro‑F1 on a held‑out test set.

  • Goal: Directly compare VADER vs a modern transformer on news headlines
  • Architecture: Serverless AWS Lambda backend + Hugging Face Inference API. You can find the model here.
  • Output: Unified JSON; DistilBERT scores mapped to a VADER‑style compound scale
  • Data: 3‑class headline sentiment (NEGATIVE/NEUTRAL/POSITIVE); consistent inputs for both models

NYC Taxi Graph Analytics Pipeline

Designed and deployed a scalable real-time graph analytics pipeline to process and analyze NYC Yellow Cab trip data.

  • Phase 1: Built a Neo4j graph database in Docker, modeled trip data as Location nodes and TRIP relationships, and implemented PageRank and BFS using the GDS library.
  • Phase 2: Scaled to a distributed, streaming architecture using Kubernetes, Apache Kafka, and Kafka Connect to ingest and update trip data in near real-time.
  • Enabled continuous computation of graph metrics (PageRank, BFS) on an ever-evolving dataset.
  • Performance: Optimized the pipeline to utilize ~70% CPU and ~65% RAM, leaving headroom for further scaling.

Tech: Neo4j, Docker, Kubernetes, Apache Kafka, Kafka Connect, Python, Graph Data Science Library.

Epileptic Seizure Detector

Developed an EEG‑based classifier to distinguish between normal brain activity and epileptic seizures. After extensive feature engineering and visualization, a combination of Random Forest and K‑Nearest Neighbor models achieved 99.2% accuracy and a recall of 0.9954. I authored a scientific paper on this work drafted for publication.

  • Processed multichannel EEG recordings and generated spectrograms for visual analysis
  • Engineered statistical and frequency‑domain features to capture seizure patterns
  • Built and evaluated multiple classifiers; selected RF + KNN ensemble for best performance

Air Quality Index (AQI) Prediction

Forecasted daily air quality for Los Angeles by combining historical AQI readings with weather data (temperature, humidity, wind speed, pressure and cloud cover). The project compared a suite of regressors and introduced feature engineering techniques such as cyclical time encoding and interaction terms. A two‑layer stacking ensemble of linear and tree‑based models reduced mean absolute error to 5.24 and achieved an R² of 0.71 on unseen data.

  • Collected data via AirNow and Open‑Meteo APIs and merged into a clean training set
  • Benchmarked models including Linear, Ridge, LGBM, XGBoost and Random Forest
  • Implemented a stacking ensemble that outperformed individual models

Movie Recommender System

Built a content‑based movie recommendation engine that suggests five similar movies based on a user’s selection. Using natural language processing to transform movie synopses into numerical feature vectors and a nearest‑neighbor model to compute similarities, the app returns titles that share themes, genres and plot elements with the chosen film. The entire application is wrapped in a sleek Streamlit interface and deployed to Heroku for easy access.

  • Parsed and vectorized movie descriptions using TF‑IDF and cosine similarity
  • Implemented a fast k‑nearest neighbours search to retrieve similar movies
  • Developed an interactive web UI in Streamlit and hosted on Heroku
